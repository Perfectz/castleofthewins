# Comparison

## Before vs After

### 1. Default creation view

Before:
- the top of the screen mixed creation with audio controls
- the right column mixed preview, full stat detail, progression context, and run advice
- mobile scan depth was high before the player could even decide on race or class

After:
- the screen opens directly into character setup
- the preview confirms the build in a shorter, quieter way
- the default surface is clearly about making a character, not reading meta systems

Key pairs:
- `screenshots/before/before-mobile-02-default.png` vs `screenshots/after/after-mobile-02-default.png`
- `screenshots/before/before-desktop-02-default.png` vs `screenshots/after/after-desktop-02-default.png`

### 2. Race and class scanning

Before:
- cards were readable but heavier than necessary
- fewer options fit comfortably in the phone viewport

After:
- cards still retain art and identity, but fit more tightly
- the player can compare race and class options with less scroll burden

Key pairs:
- `screenshots/before/before-mobile-03-race-change.png` vs `screenshots/after/after-mobile-03-race-change.png`
- `screenshots/before/before-mobile-04-class-change.png` vs `screenshots/after/after-mobile-04-class-change.png`

### 3. Progression context

Before:
- progression and contract information was always open
- the preview column felt like a second system menu

After:
- `Legacy & Contracts` becomes optional depth
- the player sees progression only when they ask for it

Key pairs:
- `screenshots/before/before-mobile-05-legacy.png` vs `screenshots/after/after-mobile-05-legacy.png`
- `screenshots/before/before-desktop-05-legacy.png` vs `screenshots/after/after-desktop-05-legacy.png`

### 4. CTA readability

Before:
- the CTA competed with too much lower-page noise

After:
- `Begin Adventure` reads more clearly as the end of the setup flow
- mobile bottom-of-screen readability is stronger because the main surface is shorter

Key pairs:
- `screenshots/before/before-mobile-06-cta.png` vs `screenshots/after/after-mobile-06-cta.png`
- `screenshots/before/before-desktop-06-cta.png` vs `screenshots/after/after-desktop-06-cta.png`
